Colour Blind Me!

The colour blindness simulator for Englishmen

View websites through the eyes of a colour blind person.

Install

Download

Add the script to the development version of your site, add the script tag below to your HTML.

<script src="colourblind.me.js"></script>

The script depends upon jQuery
