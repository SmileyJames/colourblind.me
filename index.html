Colour Blind Me The colour blindness simulator for Englishmen

View websites through the eyes of a colour blind person..

Install

Install by adding to the website code.

Add to the script to the developement version of your site. Add the below script tag to your HTML.

            
              <script src="https://colourblind.me/js"></script>

Install using developer tools.

Paste the below code into the console in the developer tools of your browser.

              (function () {
                var e = document.createElement('script');
                e.setAttribute("type","text/javascript");
                e.setAttribute("src", "https://colourblind.me/js");
                document.getElementsByTagName("head")[0].appendChild(e);
              })();
